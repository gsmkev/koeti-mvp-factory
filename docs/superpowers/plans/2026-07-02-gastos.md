# gastos — implementation plan

Spec: `docs/superpowers/specs/2026-07-02-gastos-design.md`

## Task 1 — Scaffold

`pnpm create-mvp gastos`, commit the scaffold untouched.

## Task 2 — expenses entity (crud.md recipe)

1. Schema: `expenses` table in `lib/db/schema.ts` (see spec) →
   `db:generate` + `db:migrate`, commit migration.
2. Queries: `getExpenses(teamId)` (desc by spentAt), `getMonthTotal(teamId)`
   (sum amount where spentAt in current calendar month).
3. Actions: `crudActions(expenses, { path: '/dashboard/gastos', schema })` —
   export create + remove. zod: amount `z.coerce.number().positive()`,
   category enum, description min(1), spentAt `z.string().min(1)`.
4. Page `app/(dashboard)/dashboard/gastos/page.tsx`: StatCard "Total del mes"
   - ResourcePanel (fields: amount number step 0.01, category select,
     description text, spentAt date; columns: fecha, categoría (Badge),
     descripción, monto; onDelete).
5. Nav: "Gastos" entry in `app/(dashboard)/dashboard/layout.tsx` navItems.

## Task 3 — Verify + ship

typecheck, test, build, `pnpm verify-app gastos`. Commit, push branch
`factory/gastos`, draft PR.
